相关项目

社区中有许多项目为整个社区提供基础设施和资源支持。这些项目涵盖范围广泛， 从为许多 PostgreSQL 社区活动提供活动管理软件， 到软件包管理，甚至包括 PostgreSQL.org 网站本身！

与 PostgreSQL 核心软件一样，这些项目依赖贡献者的支持来开发新功能并进行 持续维护。它们使用多种编程语言编写（在某些情况下，还涉及自然语言翻译！）， 都是人们为 PostgreSQL 社区做出有意义贡献的途径。除了每个项目本身的功能介绍外， 还提供了如何开始参与项目贡献的说明。

如果您认为某个项目应该出现在此列表中，请发送邮件至 pgsql-www@lists.postgresql.org。

网站基础设施

pgweb

"pgweb" 是 postgresql.org 网站的基础代码， 既是内容管理系统，也是许多其他 PostgreSQL 系列网站的身份认证提供方。

PostgreSQL 网站分为动态部分和静态部分：

  • 动态内容仓库
  • 静态文件仓库

该网站运行在 Django Web 框架上。您可以通过 pgsql-www 邮件列表 参与该项目。

Planet PostgreSQL

Planet PostgreSQL 是一个博客 聚合平台，汇集了人们发布的关于 PostgreSQL、PostgreSQL 社区以及 PostgreSQL 相关项目的博客内容。它包括聚合与分发软件，以及用于 辅助审核的工具。

  • Planet PostgreSQL 仓库
  • Planet PostgreSQL

您可以通过 pgsql-www 邮件列表 参与该项目。

邮件列表基础设施

pglister

PostgreSQL 邮件列表系统由 "pglister" 驱动：

  • pglister 主仓库
  • 邮件列表

pgarchives

"pgarchives" 代码库驱动 PostgreSQL 邮件列表归档的展示功能， 它是 PostgreSQL 项目开发工作流程的重要组成部分。

  • pgarchives 主仓库
  • 邮件列表归档

您可以通过 pgsql-www 邮件列表 参与这些项目。

开发工具

PostgreSQL 构建农场

PostgreSQL 构建农场在社区提供的大量设备上测试当前的 PostgreSQL 代码及所有受支持的版本：

  • 构建农场网站
  • 构建农场服务端代码仓库
  • 构建农场客户端代码仓库
  • 如何运行自己的构建农场成员
构建农场相关讨论请使用 buildfarm-members 邮件列表。

Commitfest

PostgreSQL 使用一套独特的系统来管理 "Commitfest"（提交节），即 单个开发周期。使用 Commitfest 应用的人员在 pgsql-hackers 邮件列表上讨论他们的补丁。

  • Commitfest 网站
  • pgcommitfest 主仓库

您可以通过 pgsql-www 邮件列表 参与该项目。

PostgreSQL 性能农场

PostgreSQL 性能农场用于收集性能数据：

  • 性能农场代码仓库

您可以通过 pgsql-www 邮件列表 参与该项目。

软件包管理

apt.postgresql.org

APT 仓库为基于 Debian/Ubuntu 的系统提供 PostgreSQL 软件包：

  • 包含详细信息的 Wiki 页面

您可以通过 pgsql-pkg-debian 邮件列表参与该项目。

yum.postgresql.org

YUM 仓库为基于 Red Hat/Rocky/AlmaLinux/Fedora 的系统（及其衍生版） 提供 PostgreSQL 软件包：

  • 包含详细信息的网站
  • 问题跟踪器

您可以通过 pgsql-pkg-yum 邮件列表参与该项目。

zypp.postgresql.org

Zypper 仓库为 Suse Enterprise Linux 提供 PostgreSQL 软件包：

  • 包含详细信息的网站
  • 问题跟踪器

您可以通过 pgsql-pkg-yum 邮件列表参与该项目。

活动管理

pgeu 会议系统

"pgeu-system" 为多个 PostgreSQL 会议提供支持，包括：

  • PostgreSQL 欧洲活动
  • 美国 PostgreSQL 协会活动
  • PGCon - PostgreSQL 用户与开发者大会
  • BSDCan - BSD 大会

该会议系统运行在 Django Web 框架上，是采用 MIT 许可证的开源软件。

代码托管在 git.postgresql.org 上，并镜像到 GitHub：

  • pgeu-system 主仓库
  • GitHub 镜像

GitHub 镜像可用于提交 Issue 和 Pull Request。您可以通过 pgeu-system@lists.postgresql.eu 邮件列表参与该项目。

宣传推广

新闻稿

新闻稿仓库包含 PostgreSQL 主版本发布的存档、新闻资料包及其翻译， 以及定期的小版本更新。

  • 新闻资料包仓库

如果您有兴趣成为新闻稿翻译者，可以发送邮件至 press@postgresql.org 参与该项目。
