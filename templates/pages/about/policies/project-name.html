"PostgreSQL" 与 "Postgres"

Postgres 是 PostgreSQL 项目的一个公认别名。但它只是别名或昵称， 并非项目的官方名称。引用 PostgreSQL 核心团队成员 Dave Page 的话：

在最近关于项目改名的讨论之后，很明显社区内部永远不会达成共识。 鉴于此，核心团队对此事进行了深入讨论，并决定项目将继续使用 PostgreSQL 这一名称， 但接受 Postgres 作为别名使用。

此致，Dave
