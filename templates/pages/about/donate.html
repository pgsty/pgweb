捐赠

本页列出了认可的非营利组织以及对 PostgreSQL 项目至关重要的其他非营利组织。向这些组织捐赠有助于支持我们的工作。

支持认可的非营利组织有多种方式，包括成为财务赞助者。 您可以在财务赞助页面或认可的非营利组织网站上了解更多信息。

PostgreSQL 社区协会

PostgreSQL Community Association（PGCA）是应 PostgreSQL 核心团队 的要求而设立的专用非营利组织， 负责管理和保护项目的关键资产，如域名和商标。向 PGCA 捐赠可以支持维护开源项目所需的 一般项目管理工作。

捐赠给 PostgreSQL

PostgreSQL 欧洲（PGEU）

PostgreSQL Europe 是一个欧洲地区的非营利组织，专注于支持欧洲的其他 PostgreSQL 社区团体。

捐赠给 PGEU

PostgreSQL 美国（PgUS）

PostgreSQL United States 是美国的 PostgreSQL 非营利组织，专注于教育、用户组和推广倡导。

捐赠给 PgUS

其他 PostgreSQL 工作

PostgreSQL 是 Software in the Public Interest 的附属项目。捐赠给 PostgreSQL 的资金用于赞助全球范围内的 PostgreSQL 相关工作。 这些资金由资金组管理。

捐赠给 PG @ SPI

税收减免

请注意，PostgreSQL 的捐赠可能享受也可能不享受免税待遇。如需了解更多信息， 请咨询您的捐赠对象和您的税务专业人士。

PostgreSQL 非营利组织（NPO）认可

如需了解认可的 PostgreSQL 非营利组织（NPO） 如何获得认可， 请查阅此处的指南。

问题

如果您对捐赠有任何疑问，请联系 PostgreSQL 资金组 或您所捐赠的相应组织。
