向 PostgreSQL 捐赠

PostgreSQL 的捐赠由资金小组管理。 捐赠的资金可用于 PostgreSQL 社区所需的各种服务，包括推广材料、项目基础设施、法律费用等。

通过信用卡 / Paypal 向 PostgreSQL 捐赠

您可以使用信用卡或 Paypal 账户向 PostgreSQL 社区捐赠。捐赠将存入 Software in the Public Interest（SPI）的 PostgreSQL 账户。

其他捐赠方式

PostgreSQL 是 Software in the Public Interest 的关联项目。 您可以通过 Software in the Public Interest 提供的多种方式向 PostgreSQL 捐赠，包括以下方法：

  • 通过信用卡捐赠
  • 通过 Zelle 捐赠
  • 通过 PayPal 捐赠
  • 通过支票捐赠

捐赠时，请务必在捐赠中注明指定用于 PostgreSQL。

有关上述捐赠方式以及其他捐赠方式的详细信息，请参阅 Software In The Public Interest 的 捐赠页面。

税收减免

请注意，PostgreSQL 的捐款可能享受也可能不享受免税待遇。 更多信息请参阅 SPI 网站 并咨询当地税务顾问。如果您想捐赠非货币物品（如计算机或其他设备），请联系 PostgreSQL 资金小组。

基础设施捐赠

有许多公司向 PostgreSQL 社区捐赠服务器和其他基础设施。如果您有兴趣提供基础设施，请联系 PostgreSQL 资金小组。
