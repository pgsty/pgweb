项目治理目录

本页列出了构成项目治理层级的各个团队和委员会的页面链接。请查看各个链接页面以了解每个小组的更多信息，包括成员和章程。

核心团队

PostgreSQL 核心团队（Core）是 PostgreSQL 项目政策和治理的核心仲裁者。

系统管理团队

PostgreSQL 系统管理团队（PGInfra）运行所有 postgresql.org 基础设施，包括各种公开和非公开的服务。

提交者

PostgreSQL 提交者 是拥有向 git 仓库 推送权限的人员。

安全团队

PostgreSQL 安全团队 负责监管与 PostgreSQL 及其密切相关项目有关的漏洞报告和修复工作。如果发现 PostgreSQL 或从 PostgreSQL 下载页面 链接的任何安装程序中的安全漏洞，请发送电子邮件至 security@postgresql.org。如需报告非安全性 Bug，请访问报告 Bug 页面。如果您不确定某个问题是否属于安全漏洞，请谨慎起见，发送电子邮件至 security@postgresql.org。

行为准则委员会

行为准则委员会负责接收和调查所有与 PostgreSQL 行为准则 相关的投诉。

贡献者委员会

贡献者委员会负责维护贡献者档案页面。

赞助者委员会

赞助者委员会负责维护赞助者页面。

非营利组织

多个非营利组织以各种方式为 PostgreSQL 项目提供服务。
