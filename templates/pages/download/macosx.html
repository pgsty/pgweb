macOS 安装包

您可以从多个来源获取 macOS PostgreSQL 安装包。

EDB 交互式安装程序

下载安装程序，由 EDB 认证，支持所有受支持的 PostgreSQL 版本。

注意！此安装程序托管在 EDB 上，而非 PostgreSQL 社区服务器上。如果您在托管网站上遇到问题，请联系 webmaster@enterprisedb.com。

此安装程序包含：

  • PostgreSQL 服务器
  • pgAdmin，用于管理和开发数据库的图形化工具
  • StackBuilder，用于下载和安装额外 PostgreSQL 工具和驱动程序的包管理器。StackBuilder 包含管理、集成、迁移、复制、地理空间、连接器和其他工具。

此安装程序可以在图形化、命令行或静默安装模式下运行。

macOS 安装程序旨在提供一种简单、快速的方式来安装和运行 PostgreSQL。如果您在安装程序方面遇到问题（这种情况不太可能发生），请在 安装程序 Github 页面 上报告。已安装软件包的问题应直接向相应项目报告。

高级用户还可以下载不含安装程序的 zip 压缩包（仅二进制文件）。此下载适用于希望将 PostgreSQL 打包进其他应用程序安装程序的用户。

平台支持

安装程序由 EDB 在以下平台上测试。通常也可以在更新版本的 macOS 上运行：

PostgreSQL 版本 64 位 macOS 平台
18 13+ (amd64), 13+ (arm64)
17 11.x (amd64), 12.x (arm64)
16 11.x (amd64), 12.x (arm64)
15 10.14 - 12.x (amd64), 12.x (arm64)
14 10.14 - 12.x (amd64), 12.x (arm64)
13 10.14 - 11.0

Postgres.app

Postgres.app 是一个简单的原生 macOS 应用程序，在菜单栏中运行，无需安装程序。打开应用程序，即可获得一个就绪并等待新连接的 PostgreSQL 服务器。关闭应用程序，服务器即会关闭。

Homebrew

也可以使用 Homebrew 在 macOS 上安装 PostgreSQL。例如，要安装 PostgreSQL 15，可以使用以下命令：

brew install postgresql@15

您可以使用 Homebrew Formulae 搜索 PostgreSQL 和其他软件包。要查找特定的 PostgreSQL 主版本，请搜索 postgresql@NN，其中 NN 代表主版本号。例如，postgresql@15。

MacPorts

也可以从 MacPorts Project 获取 macOS 上的 PostgreSQL 软件包。请参阅 MacPorts 文档以了解如何安装 ports。

可以在 MacPorts 网站上使用 portfiles 搜索工具查找 PostgreSQL 软件包 列表。

Fink

可以从 Fink Project 获取 macOS 上的 PostgreSQL 软件包。请参阅 Fink 文档以了解如何安装软件包。

可以在 Fink 网站上使用软件包搜索工具查找 PostgreSQL 软件包 列表。
