Linux 下载（SUSE）

所有 SUSE 版本默认都提供 PostgreSQL。但是，SUSE Linux 会“快照”一个特定版本的 PostgreSQL，并在该 SUSE 版本的整个生命周期内提供支持。

PostgreSQL RPM Repository

该仓库将与您的常规系统和补丁管理集成，并在 PostgreSQL 的整个支持生命周期内为 PostgreSQL 13+ 提供自动更新。

PostgreSQL RPM 仓库目前支持 SUSE Enterprise Linux 15 SP7。同时也为 SLES 12 SP5 更新 PostgreSQL 软件包。

要使用 SUSE RPM 仓库，请按照此说明操作。

发行版自带

SUSE Linux 和 openSUSE 的 RPM 包可从 openSUSE Build Service 的 server:database:postgresql 项目中获取。 提供了针对特定平台的 PostgreSQL RPM 包以及各种相关软件。 请使用搜索功能查找合适的软件包。文档也可在该处获取。
