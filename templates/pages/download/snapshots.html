Beta/RC 版本及开发快照（不稳定）

Beta 和候选发布（Release Candidate）安装包在 PostgreSQL 新主要版本发布之前构建，不会持续构建。

开发快照基于开发者正在开发的最新源代码和 Beta/RC 版本构建。 这些是 PostgreSQL 的"前沿"版本， 不应在生产系统中使用，因为它们几乎没有经过测试或质量控制。

最新的开发版文档也在线可用。

源代码

在 PostgreSQL 新主要版本开发周期的最后阶段，Beta 和候选发布的源码包会与发布版源码包一同提供。

源代码压缩包每晚在 PostgreSQL 官方开发服务器上自动构建。开发快照取自 git 仓库的最新提交，包含正在为下一个版本开发的新功能。还有从所有受支持的稳定分支构建的"分支最新"快照，其中包含计划在下一个版本中发布的所有错误修复。

Red Hat、Rocky Linux、AlmaLinux 和 Fedora

适用于 Red Hat、Rocky Linux、AlmaLinux 和 Fedora 的 RPM 包可从 PostgreSQL Yum 仓库获取。

要为这些版本设置仓库，请按照这些平台上常规 Yum 仓库的相同说明操作。

Debian 和 Ubuntu Linux

适用于 Debian 和 Ubuntu 的 DEB 包可从 PostgreSQL APT 仓库获取。

要为这些发行版设置仓库，请按照 Linux 下载（Debian）或 Linux 下载（Ubuntu）上的说明操作。

安装程序

适用于 Windows 和 Mac 的安装程序可在此处获取（外部链接）。这些安装程序还包含 pgAdmin，由 EDB 发布。

仅 Beta 和候选发布版本提供安装程序。
