Windows 安装程序

EDB 交互式安装程序

下载安装程序，由 EDB 认证，支持所有受支持的 Postgres 版本。

注意！此安装程序托管在 EDB 上，而非 PostgreSQL 社区服务器上。如果您在托管网站上遇到问题，请联系 webmaster@enterprisedb.com。

此安装程序包含：

  • PostgreSQL 服务器
  • pgAdmin，用于管理和开发数据库的图形化工具
  • StackBuilder，用于下载和安装额外 PostgreSQL 工具和驱动程序的包管理器。StackBuilder 包含管理、集成、迁移、复制、地理空间、连接器和其他工具。

此安装程序可以在图形化或静默安装模式下运行。

Windows 安装程序旨在提供一种简单、快速的方式来安装和运行 Postgres。如果您在安装程序方面遇到问题（这种情况不太可能发生），请在 安装程序 Github 页面 上报告。已安装软件包的问题应直接向相应项目报告。

高级用户还可以下载不含安装程序的 zip 压缩包（仅二进制文件）。此下载适用于希望将 Postgres 作为其他应用程序安装程序一部分的用户。
