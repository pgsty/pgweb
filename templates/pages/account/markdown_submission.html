Markdown 提交

postgresql.org 网站上的部分表单允许您以 markdown 格式提交文本。这是一种简单的标记格式，可以让您进行基本的内联排版。

您可以在 markdownguide.org 上找到 markdown 的基本介绍。

本站支持 基本 markdown 语法，但有以下例外：

  • 不允许使用图片
  • 不允许使用相对链接，仅允许绝对链接
  • 不允许链接到 http、https 或 mailto 以外的协议。
  • 可以使用内联 HTML，但只允许使用与 markdown 生成的相同标签和属性。强烈建议使用原生 markdown。

预览

markdown 字段的实时预览在 Internet Explorer 上无法使用。

预览是异步更新的，因此在进行更改后，预览更新可能会有几秒钟的延迟，具体取决于网络速度。
